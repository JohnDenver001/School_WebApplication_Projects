# FirstYear_WebApplication_Project

A static e-commerce website for a tech-enhanced clothing brand built with HTML, CSS, and JavaScript.

## Project Description

FirstYear_WebApplication_Project is a front-end web application that showcases Pasek' Clothing Tech, a fictional brand specializing in technology-integrated fashion products. The project demonstrates fundamental web development skills including responsive design, interactive UI components, and modern CSS styling techniques. This project was developed as a first-year school assignment to demonstrate proficiency in HTML, CSS, and JavaScript.

## Features

- Responsive navigation with sticky header
- Hero section with promotional content
- Featured products showcase with pricing and descriptions
- Interactive FAQ section with expandable answers
- Contact form with form validation
- User login and signup pages
- Shop page displaying product catalog
- About page with company information
- Smooth scrolling navigation
- Add to cart functionality with user feedback

## Tech Stack

### Frontend

- HTML5
- CSS3
- JavaScript (Vanilla)

### Tools

- Modern web browser for development and testing

## Installation and Setup

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, or Safari)
- A text editor or IDE (VS Code recommended)
- Local web server (optional, for development)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/JohnDenver001/School_WebApplication_Projects.git
   ```

2. Navigate to the project directory:
   ```bash
   cd School_WebApplication_Projects/FirstYear_WebApplication_Project
   ```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local development server

## Usage

1. Open `index.html` in a web browser to view the homepage.
2. Use the navigation menu to browse different sections:
   - Home: View featured products and FAQ
   - About: Learn about the company
   - Contact Us: Send a message through the contact form
   - Shop: Browse the product catalog
3. Click "Login" to access the login page.
4. Click "Add to Cart" buttons to simulate adding products to cart.
5. Expand FAQ items by clicking on the questions.

## API Documentation

This project does not include a backend API. All functionality is handled client-side using JavaScript.

## Testing

This project does not include automated tests. Manual testing can be performed by:

1. Opening each HTML page in a web browser
2. Verifying all links navigate correctly
3. Testing form submissions
4. Checking responsive behavior at different screen sizes

## Known Issues and Limitations

- No backend integration for user authentication
- Cart functionality is simulated with alerts only
- Contact form does not send actual emails
- Product images may require local files to display correctly
- No persistent data storage
